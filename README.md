# frontend-v3

Biblioteca de componentes Vue 3.

[Exemplo](https://danielthavares.github.io/frontend-v3/)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
